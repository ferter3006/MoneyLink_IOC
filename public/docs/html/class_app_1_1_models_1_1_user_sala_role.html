Detailed Description

Class UserSalaRole Clase que representa las instancias de usuario en cada sala y rol que tienen en esa sala.

La Clave primaria es el conjunto de user_id, sala_id y role_id. Para que no pueda haber dos instancias de un usuario en la misma sala.
